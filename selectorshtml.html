Different Types of Selectors:

Type selectors- Any HTML element can be a type selector, e.x. article, header, h1, nav etc. CSS can be applied to any element and all occurances of that element.

Class selectors- Unlike an id, a class attribute can be applied to many occurances of a specific element, the element belongs to the class. For example, the class "highlight" has been used throughout (the class "highlight" seperates the sections where they are developed in css to have a pink background) and a selector would select all of the elements with a class value, highlight allowing for CSS rules to be applied to the whole section of elements.

ID selectors- Can have different values if the user declares them seperately. For example, this sections id is "id_selector" which can be uniquely selected as it's id has unique values.

Psuedo selectors- Used to define a special state of an element. For example, it can be used to style an element when a user mouses over it, style visited and unvisited links and style an element when it gets clicked on -a:link, a:visited, a:hover, a:active-

Selector Combination Based on the DOM Relationship- A combination is something that explains the relationship between the selectors, a CSS selector can contain more than one simple selector. There are four different combinators in CSS:

  • Descendant selector (space)
  • Child selector (>)
  • Adjacent sibling selector (+)
  • General sibling selector (~)

    Extra Information:
  • Selectors can be combined! -footer > h1 selects only the h1 element that is the child of the footer element-
  • Selectors can be specified as a comma seperated list to have the same CSS rule! -div, h1 { color: red;} selects all div and h1 elements, whereas without a comma this selects only the h1 within the div element.
