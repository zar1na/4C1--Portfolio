Structure answers:

A) What is a markup? A markup is a way to define and individualise elements.

B) What is hypertext and hypermedia? Hypertext is a link to other texts and hypermedia is similar but also includes graphics, sound and video.

C) What is a doctype and why is it important to include it in every HTML document? A doctype specifies the version of HTML and therefore should be included in every HTML document, the browser will attempt to render the document without a doctype (called "quirck mode") with unexpected results.

D) What language can be used to style web documents? HTML is a language that can be used to style web documents.

E) Which elements are required for a valid XHTML5 document? For a valid XHTML5 document elements must: be properly nested, have a start and end tag and be carefully case-sensitive, attribute values: must be enclosed in double quotation marks and may not be repeated.

F) What is a self closing HTML element? Give an example? Any tag can be self-closed by the ending slash <

G) What is XML and why is it important to web developers? XML- the X in XHTML is an abbreviation for XML which is a general purpose markup language with stricter rules.

H) Describe 4 XML rules? XML rules: elements must: be properly nested, have a start and end tag and be carefully case-sensitive, attribute values: must be enclosed in double quotation marks and may not be repeated.

I) What is the difference between a "well-formed" and a valid web document? A document that conforms to a particular doctype is a "wellformed" pre-structure, a valid web document.
